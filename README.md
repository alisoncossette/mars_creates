# mars_creates

**Mars, the robot content creator** ([@littlebotmars](https://x.com/littlebotmars)). An Innate MARS
agent that roams an event, asks people what they're working on, and posts a shout-out to **X** —
with an image generated by **Magnific**.

## How it works (Innate-native)

The Innate cloud brain runs the conversation, voice, and the loop. The post is built by a custom skill.

```
agents/content_creator.py   -> robot ~/innate-os/agents/   (the Agent: prompt + skills + ["micro"])
skills/publish_interview.py -> robot ~/skills/             (local/publish_interview)
skills/take_selfie.py       -> robot ~/skills/             (local/take_selfie — optional photo)
mars/                       -> robot ~/mars_lib/           (lib: config, content, publish, llm)
run_mars.py                 -> off-robot mock harness (play the whole flow on a laptop)
```

## The loop

```
roam -> consent -> ask @handle + "what are you working on?" -> consent to post
     -> Magnific generates an image from what they said
     -> Claude writes the caption
     -> post to X, tagging the person + sponsors
```

## Run the mock (off-robot, no keys)

```bash
python run_mars.py --rounds 1   # type the guest's answers; everything mocks gracefully
```

## Keys (`~/mars_lib/.env` on the robot)

| Var | For |
|---|---|
| `X_API_KEY` / `X_API_SECRET` / `X_ACCESS_TOKEN` / `X_ACCESS_SECRET` | posting to X |
| `MAGNIFIC_API_KEY` | the generated image (optional → text-only) |
| `ANTHROPIC_API_KEY` | Claude caption (optional → template) |

## Deploy

1. `content_creator.py` → robot `~/innate-os/agents/`
2. `skills/*.py` → robot `~/skills/`
3. `mars/` → robot `~/mars_lib/mars/`  (and `pip install anthropic tweepy python-dotenv httpx`)
4. keys in `~/mars_lib/.env`, then start **Content Creator** in the Innate app.
